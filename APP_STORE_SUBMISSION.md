```
This app is a web-based iOS wrapper (WKWebView) loading https://smarttechbazaar.com — a B2B and B2C e-commerce platform for computer accessories and IT solutions based in Bangalore, India.

NAVIGATION: All navigation is done through the in-app web interface. The app displays a native loading bar and supports iOS gestures.

PAYMENTS: The checkout flow uses Razorpay (a PCI-DSS compliant Indian payment gateway). To test, use the demo account above and place an order — you can abandon the payment step; you do not need to complete a real transaction.

DEMO ACCOUNT: Log in with reviewer@smarttechbazaar.com / AppReview@2026. The account has pre-loaded order history and a wishlist so all features can be reviewed without making a purchase.

B2B PRICING: Some products show different prices for B2B accounts. The demo account is a standard B2C account so all prices shown are retail prices — this is expected behaviour.

LOCATION / REGION: The app is intended for India. All prices are in Indian Rupees (INR / ₹). Shipping is available across India only.
```
